# Mkdocs - GitBook Theme

[![Apache 2.0 License][apache-badge]](LICENSE)
[![PyPI][pypi-badge]][pypi-ref]

## Installation

First, install the package via PyPI:

```sh
pip install mkdocs-gitbook
```

Then include the theme in your `mkdocs.yml` file:

```yaml
theme:
  name: gitbook
```
